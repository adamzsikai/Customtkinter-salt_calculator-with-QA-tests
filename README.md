# Customtkinter-salt_calculator-with-QA-tests
The repository contains:  salt_calculator.py -> Customtkinter based GUI app for calculating a powderblend and a certain additional component overall salt content after mixing.
                          salt_calculator_cli.py -> A simplier CLI version.
                          Manual test cases (documented in jira).
                          Refactored code (suitable for unit testing).
                          Automated unit test with pytest module (documented in jira).
                          Automated integration test for CLI and GUI ( documented in jira).
